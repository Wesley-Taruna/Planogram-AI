# YOLOv8 Test Suite — FamilyMart Planogram

Completely isolated from the main API. Run these scripts to test
and train YOLOv8 before any integration.

---

## The 5-Step Journey

```
STEP 1  → See what pre-trained YOLO detects (no training needed)
STEP 2  → Generate your class list + understand labeling format
STEP 3  → Train a custom model on your labeled shelf photos
STEP 4  → Test your trained model on a shelf photo
STEP 5  → Preview full hybrid compliance output (YOLO + planogram JSON)
```

---

## Step 1 — Pre-trained detection (run this first)

```bash
python yolo_test/1_test_pretrained.py --photo <your_shelf_photo.jpg>
```

**What you'll see:** Generic object detection (bottle, box, cup).
This shows WHY custom training is needed.
Output: `yolo_test/results/pretrained_result.jpg`

---

## Step 2 — Generate class list from your planograms

```bash
python yolo_test/2_label_guide.py
```

**What you'll see:** Full labeling instructions + your product class list.
Output: `yolo_test/training_data/classes.txt` + `dataset.yaml`

---

## Step 3 — Label your photos (external tool)

Before training, you need labeled shelf photos.

**Easiest option — Roboflow (free):**
1. Go to https://roboflow.com → New Project → Object Detection
2. Upload 50–100 shelf photos
3. Draw bounding boxes around each product, assign the product name
4. Export as **YOLOv8 format**
5. Put images in `yolo_test/training_data/images/train/`
6. Put labels in `yolo_test/training_data/labels/train/`

**Offline option — LabelImg:**
```bash
pip install labelImg
labelImg
```
Set format to YOLO, save to `yolo_test/training_data/labels/`

---

## Step 3 — Train

```bash
# Quick test (fast, less accurate)
python yolo_test/3_train.py --model yolov8n --epochs 50

# Recommended for real use
python yolo_test/3_train.py --model yolov8s --epochs 100
```

Output: `yolo_test/results/train/weights/best.pt`

---

## Step 4 — Test trained model

```bash
python yolo_test/4_test_trained.py --photo <shelf_photo.jpg>
```

Output: `yolo_test/results/trained_result.jpg` + detections JSON

---

## Step 5 — Full hybrid preview

```bash
python yolo_test/5_compare_with_planogram.py \
    --photo <shelf_photo.jpg> \
    --planogram SNACK3C
```

Output: Full compliance report (correct / missing / misplaced / unexpected)

## How many photos do you need to label?

| Photos labeled | Expected accuracy |
|---|---|
| 0 | Pre-trained only (detects "box", not "Chitato") |
| 50–100 | Rough detection, ~60–70% accuracy |
| 200–500 | Good detection, ~80–90% accuracy |
| 500+ | Production quality, >90% accuracy |

Minimum recommended: **200 photos**, at least **20 per product**.
